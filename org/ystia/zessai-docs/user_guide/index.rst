Big Data Capabilities Framework
###############################

**Big Data Capabilities Framework** (BDCF) is a composable **Platform as a Service** (PaaS) designed to ease the development and deployment of enterprise-grade Big Data applications.

**BDCF** provides software components and application topology templates that conform with the TOSCA standard.

It also provides tools for the development of Big Data applications, their deployment to the Cloud, and for monitoring them.
